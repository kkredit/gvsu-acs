CIS 658 Final Coding

Option 1: JavaScript

Build a client-side calendar in JavaScript. Specifically,

  • Complete calendar.js so that:
    • It updates the table in calendar.html to reflect the desired month.
    • It advances to the previous/next month when the arrows are clicked.
    • It displays the requested month when the "Submit" button is pressed.
  • Write a CSS file named calendar.css to format the calendar nicely.
    • Add a border around each day.
    • Add a border around the whole table.
    • Size the arrows nicely.
    • Size the table caption nicely.
    • Add your own personal touch
Hints
  • Remember the getElementsByTagName method.
  • Months begin with 0, not 1.
  • "&nbsp;" stands for "nonbreaking space". I used it in place of a simple " " because not all browsers will draw an empty <td> element.
  • A link to the JavsScript Date API

Option 2: MadLibs

Build a small web app (using Rails, React, Node, PHP, or any other framework you like) to play MadLibs:
  1. Display a web page with a form asking for general terms like "a noun" "a verb", "a place", "an adjective".
  2. When the form is submitted, the app should respond with the filled out story
The story should be very nicely formatted using CSS.
  • Decorate the user-specifed words in some way.
  • Add some kind of border or other decoration to the story.
  • Add your own personal touch.
